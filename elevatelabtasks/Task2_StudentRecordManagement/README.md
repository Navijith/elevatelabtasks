# Student Record Management System

## Objective

A command-line based Student Record Management System developed in Java that performs CRUD (Create, Read, Update, Delete) operations using an ArrayList.

## Features

- Add Student
- View All Students
- Update Student Details
- Delete Student
- Prevent Duplicate Student IDs
- Menu-Driven Console Interface

## Technologies Used

- Java
- OOP (Classes & Objects)
- ArrayList
- Scanner
- Encapsulation

## Project Structure

```

```

## How to Compile

```bash
javac Student.java StudentManagementSystem.java
```

## How to Run

```bash
java StudentManagementSystem
```
