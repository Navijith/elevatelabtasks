# Java Console Calculator

## Objective
A simple console-based calculator built using Java.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Handles divide-by-zero
- Runs continuously until Exit is selected

## Technologies

- Java
- Scanner
- Methods
- Loops
- Switch Case
- Exception Handling

## How to Run

Compile

javac Calculator.java

Run

java Calculator
